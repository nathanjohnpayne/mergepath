Replay your real PRs

    

      
This page is static HTML—it can't shell out to gh from the browser. A tiny helper script does it for you: it lists your repo's recent merged PRs, bakes the data into a copy of this file, and opens the copy in a new tab.

      
    
        
  1. Open a terminal at your repo root (the folder with .github/review-policy.yml).
    2. 
        
  2. Run the helper. Default pulls the last 20 merged PRs:
          
    $./scripts/policy-sim.sh
    
          Or pass a custom limit (any positive integer):
          
    $./scripts/policy-sim.sh 50
    
        
    3. 
        
  3. A new browser tab opens with the same dashboard, but the live badge will be lit and the routing simulation will replay your real merged PRs under the current policy.
    4. 
      

    

    

      Needs gh, jq, and python3 on PATH. gh must be authenticated (gh auth status). Nothing is written back to your repo—the baked copy lives in a temp file.
